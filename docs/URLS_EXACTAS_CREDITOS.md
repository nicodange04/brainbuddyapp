# 🔗 URLs Exactas para Agregar Créditos a Gemini

## 🎯 URLs Directas (Copia y Pega)

### Opción 1: Google AI Studio - Billing (Principal)

**URL Directa:**
```
https://aistudio.google.com/settings/billing
```

**Si no funciona, prueba estas alternativas:**

```
https://aistudio.google.com/app/settings/billing
https://aistudio.google.com/settings
https://aistudio.google.com/app/settings
```

---

### Opción 2: Google Cloud Console - Billing (Alternativa)

**URL Directa:**
```
https://console.cloud.google.com/billing
```

**Si necesitas crear una cuenta de facturación:**
```
https://console.cloud.google.com/billing/create
```

**Si necesitas vincular una cuenta existente:**
```
https://console.cloud.google.com/billing/linked
```

---

### Opción 3: Google Cloud Console - Seleccionar Proyecto Primero

**Si no tienes proyecto:**
1. Primero ve a: `https://console.cloud.google.com/`
2. Click en el selector de proyectos (arriba)
3. "New Project"
4. Luego ve a: `https://console.cloud.google.com/billing`

## 📝 Pasos con URLs Exactas

### Método 1: Desde Google AI Studio

1. **Abre esta URL:**
   ```
   https://aistudio.google.com/settings/billing
   ```

2. **Si te pide iniciar sesión:**
   - Inicia sesión con la cuenta que tiene tu API key

3. **Si ves "Add payment method" o botón similar:**
   - Click ahí y agrega tu tarjeta

4. **Si NO ves la opción de billing:**
   - Ve al Método 2 (Google Cloud Console)

---

### Método 2: Desde Google Cloud Console (Más Confiable)

1. **Primero, asegúrate de tener un proyecto:**
   ```
   https://console.cloud.google.com/
   ```
   - Si no tienes proyecto, créalo:
     - Click en el selector de proyectos (arriba)
     - "New Project"
     - Nombre: "Brain Buddy"
     - Click "Create"

2. **Luego ve a Billing:**
   ```
   https://console.cloud.google.com/billing
   ```

3. **Si no tienes cuenta de facturación:**
   ```
   https://console.cloud.google.com/billing/create
   ```
   - Completa el formulario
   - Agrega tu tarjeta
   - Vincula el proyecto

4. **Habilita la API de Gemini:**
   ```
   https://console.cloud.google.com/apis/library/generativelanguage.googleapis.com
   ```
   - Click en "Enable" o "Habilitar"

## 🚨 Si Ninguna URL Funciona

### Verifica que estás en la cuenta correcta:

1. **Cierra todas las sesiones:**
   ```
   https://myaccount.google.com/
   ```
   - Ve a "Security" → "Your devices"
   - Cierra sesiones si es necesario

2. **Inicia sesión específicamente:**
   - Abre ventana de incógnito (Ctrl+Shift+N)
   - Ve a: `https://aistudio.google.com/`
   - Inicia sesión con la cuenta correcta
   - Luego intenta: `https://aistudio.google.com/settings/billing`
